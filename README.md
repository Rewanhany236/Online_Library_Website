Online Library Website

Project Overview

The Online Library is a web application designed to help users browse, search, and borrow books easily.
The website provides a simple and user-friendly interface where users can explore available books, view detailed information, and borrow books when they are available.

This project was developed as part of the IS231 – Web Technology course (Phase 1).

Website Features

User Features

- Create an account through the Sign Up page
- Log in to the website
- Search for books by title, author, or category
- View a list of available books
- View detailed information about each book
- Borrow available books
- View the list of borrowed books

Admin Features

- Add new books to the library
- Edit book information
- Delete books from the system
- View all books in the library

---

Website Pages

The project includes the following main pages:

- home.html – Homepage
- signup.html – User registration page
- login.html – Login page
- books.html – Browse and search books
- book-details.html – Book details page
- borrowed-books.html – List of borrowed books
- admin-dashboard.html – Admin control panel
- add-book.html – Add or edit book page
